### The Hard Method
```js
GET QR
$ apt update
$ apt install nodejs --fix-missing
$ pkg install git
$ git clone https://github.com/shahmal/Ninja
$ cd Ninja
$ chmod +x *
$ npm install @adiwajshing/baileys
$ npm install chalk
$ node qr.js

SETUP
$ git clone https://github.com/shahmal/Ninja
$ cd Ninja
$ chmod +x *
$ npm i
$ node qr.js
   // scan the qr using whatsapp web on your phone
$ node bot.js
```
